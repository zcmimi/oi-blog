[置底]LG 3515 [POI2011]Lightning-Conductor
四边形不等式动态规划单调队列决策单调性
zc
2019-12-21 19:47:00
  • 刷题记录
查看原题

点击跳转

题目大意：

给定一个长度为 n\leq 500000 的序列 a_1, a_2, \cdots, a_n ,要求对于每一个 1 \leq r \leq n ,找到最小的非负整数 f_{r} 满足

\forall l\in\left[1,n\right]:a_l \leq a_r + f_{r} - \sqrt{|r-l|}

解法

考虑l<r,r>l的将序列翻转后再来一次就可以了

f_r= \max(a_l+\sqrt{r-l})-a_r(l\in[1,r])

可以证明f_r的最优决策点是递增的

可以打表

分治解法

[置底]LG 5504 [JSOI2011]柠檬
动态规划决策单调性单调队列
zc
2019-12-21 19:47:00
  • 刷题记录
查看原题

点击跳转

设f_i表示[1,i]最多变成多少个

f_i=\max_{s_i=s_j}(f[j-1]+s_j\times (\sum_{i=j}^i[s_i=s_j])^2)

我们可以发现如果s_j=s_{j'}(j'< j),从j'转移一定比从j优

这一条件满足决策单调性！

我们可以用单调队列维护单调性
